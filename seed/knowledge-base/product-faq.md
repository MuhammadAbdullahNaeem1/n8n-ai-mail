# Flowdesk — Product FAQ

## What is Flowdesk?
Flowdesk is a project and client management platform for small teams (5–100 people). Web app + mobile apps (iOS/Android).

## Integrations
- **Native:** Slack, Google Workspace, Microsoft 365, Stripe, Zapier, Shopify (orders sync to projects as tasks).
- **CSV import:** customers, contacts, and tasks can be imported from CSV (Settings → Import). Max 50,000 rows per file.
- **API:** REST API + webhooks on Pro and Team plans.

## Mobile offline mode
Yes. The mobile app queues all changes made offline and syncs automatically when connectivity returns. Conflicts are resolved last-write-wins with a visible conflict log. Offline mode is available on all plans.

## Data export
- Individual project export: any plan, Settings → Export.
- **Bulk export (all projects, single zip): available on Team plan** or on request via support for compliance/audit purposes — we can generate a full activity-log export (up to 24 months) within 1 business day.

## Plans & pricing
- **Starter:** $12/user/month — core features.
- **Pro:** $24/user/month — API access, integrations, priority email support.
- **Team:** $39/user/month — SSO/SAML, bulk export, audit logs, priority support with 4-hour response SLA, dedicated onboarding.
- Annual billing: 2 months free (≈17% off).
- Volume discounts start at 20 seats — sales can quote custom annual pricing.

## Trials
- 14-day free trial of the Team plan, no credit card required.
- **Trial extensions:** one-time 7-day extension available on request — just reply and ask.
